# kottans-frontend 
Цікавий матеріал по Git/Имеет решающее значение для поддержания здоровой кодовой базы для всех видов ИТ-ресурсов,и для того, чтобы несколько людей сотрудничали в одних и тех же проектах кодирования вместе./
можем использовать инструмент командной строки diff, чтобы взять два файла или даже в каталоги, и показать различия между ними в нескольких форматах.
